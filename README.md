# Tickoffee
Software que permite realizar pedidos de platillos y bebidas, seleccionando cantidades y métodos de pago generando tickets.


"Objetivo"
El software de la cafeteria va a tomar el pedido, lo va a guardar y lo confirmara para
después mostrar un ticket sobre su compra.
Esto con el fin de ayudar a los empleados de la cafeteria a no olvidar los pedidos de los alumnos.
